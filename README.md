## General info
This project is a simple web service, exposed via REST APIs, that allow SMEs keep secure ledgers,
taking advantage of a simple public blockchain.

## Technologies
Project is created with:
* Python 3
* Flask version: 1.1.1
* HTML/CSS
*REST APIs
